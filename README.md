# Task 8 – Simple Sales Dashboard (Power BI)

## Objective

Create a basic interactive dashboard that shows sales performance by product, region, and month.

## Tools Used

* Power BI
* Superstore Sales Dataset

## Dashboard Visualizations

1. **Line Chart** – Sales over Months
2. **Bar Chart** – Sales by Region
3. **Donut Chart** – Sales by Category
4. **Slicer** – Region filter

## Insights

* The **West region generated the highest sales**.
* **Technology category contributes the largest share of revenue**.
* Sales vary across months showing **seasonal trends**.
* The **South region shows lower sales performance** compared to others.

## Dataset

Superstore Sales Dataset
